Taller de Programación WebEste proyecto consiste en el desarrollo de un sitio web llamado Hotel BioTalca. El sitio ha sido construido utilizando HTML5 y Bootstrap 5. Estructura del ProyectoEl sitio web consta de las siguientes secciones:
Navbar Fija: Navegación fluida con logo, menú y botón de reserva destacado.Hero
Carousel: Presentación visual con 3 capturas de pantalla de alta calidad y texto superpuesto.
Habitaciones: Sección con 4 tipos de habitaciones (Standard Single, Standard Doble, King Matrimonial y Penthouse) utilizando el sistema de Cards.
Servicios: Listado de beneficios del hotel mediante el uso de íconos.
Galería: Imágenes implementadas con row-cols y espaciado g-3, incluyendo efectos visuales de hover.Sección Destacada: "Por qué elegirnos" con una estructura de 3 columnas.
Contacto: Información detallada, datos de contacto y mapa interactivo (iframe) centrado en las coordenadas 35°43'51.1"S 71°41'32.9"W.
