Taller de Programación WebEste proyecto consiste en el desarrollo de un sitio web llamado Hotel BioTalca. El sitio ha sido construido utilizando HTML5 y Bootstrap 5. Estructura del ProyectoEl sitio web consta de las siguientes secciones:
Navbar Fija: Navegación fluida con logo, menú y botón de reserva destacado
Carousel: Presentación visual con 3 capturas y texto superpuesto.
Habitaciones: Sección con 4 tipos de habitaciones (Standard Single, Standard Doble, King Matrimonial y Penthouse) utilizando el sistema de Cards.
Servicios: Listado de beneficios del hotel mediante el uso de íconos.
Sección Destacada: "Por qué elegirnos" con una estructura de 3 columnas.
Contacto: Información detallada, datos de contacto y mapa interactivo
